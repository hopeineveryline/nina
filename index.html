Friendly NixOS management, instantly installable.

Nina gives NixOS users a warm, terminal-first CLI and TUI for rebuilds, package search, generation history, diagnostics, and remote workflows. If someone lands here and wants Nina, the install path should be one click away.

Run it right now
nix run github:asha-software/nina
Install it into your profile
nix profile install github:asha-software/nina

Search

Search nixpkgs in a TUI, inspect details, then install or try packages without leaving Nina.

History

Browse generations interactively, diff them, and jump between known-good states with calm guidance.

Doctor

Get machine summaries, diagnostics, and friendlier repair hints for the rough edges of NixOS life.
